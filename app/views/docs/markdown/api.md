# API

> **Status: Coming soon.** A public REST API for QUL resources is in active design and not yet available.

QUL is currently a downloadable resource platform — the recommended way to use the data today is to [download datasets](download-resources.md) in JSON or SQLite and integrate them directly.

## What's Planned

A single, comprehensive REST API that exposes every Quranic resource managed by QUL: the Quran text in every script variant, translations, tafsirs, footnotes, transliterations, recitations and audio segments, mushaf layouts, fonts, morphology, topics, themes, similar ayahs, mutashabihat, and surah metadata.

```
https://qul.tarteel.ai/api/v1
```
