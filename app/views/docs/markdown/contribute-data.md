# Contribute Data

QUL is more than a download platform — it is a community effort to build accurate, well-structured Quranic datasets. You can help correct, refine, and expand resources such as translations, tafsirs, transliterations, word morphology, and audio segments.

> This section is being expanded. If you contribute data regularly and want to document the workflow, please open a pull request — see [Contribution Workflow](contributing.md).

## Ways to Contribute Data

- **Proofread and correct** existing translations, tafsirs, and transliterations.
- **Segment and verify** recitation audio against ayah boundaries.
- **Refine linguistic data** — word morphology, roots, lemmas, and parts of speech.
- **Tag and relate** ayahs by topic, theme, similarity, and mutashabihat.

## How It Works

Most data contribution happens through the QUL CMS, where each resource type has its own editing and review workflow. Contributions are reviewed before they are published to the resources directory.
